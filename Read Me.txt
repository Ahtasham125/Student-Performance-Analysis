# Student Performance Analysis using PySpark

## Project Overview
This project analyzes student academic performance using PySpark.

## Features
- Data loading using PySpark
- Student marks analysis
- Average score calculation
- Top and low performers identification
- Subject-wise performance analysis

## Technologies Used
- Python
- PySpark
- Jupyter Notebook
- Pandas

## Dataset
Sample student dataset in CSV format.
